# Placeholder for project screenshots

Place your project screenshots in this folder.

## Naming Convention

Use descriptive names for your screenshots:
- `project-name-1.png`
- `project-name-2.png`
- `project-name-home.png`
- `project-name-dashboard.png`

## Recommended Sizes

- Width: 1200px - 1920px
- Height: Maintain aspect ratio
- Format: PNG or JPG
- Optimize images before uploading

## Using Screenshots

Reference screenshots in `src/data/projects.ts`:

```typescript
screenshots: [
  '/screenshots/bcr-api-1.png',
  '/screenshots/bcr-api-2.png'
]
```
